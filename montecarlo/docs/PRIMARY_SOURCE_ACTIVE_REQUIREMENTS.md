# Baseline Primaire - Active Requirements

> Date de creation : 2026-03-23
> Statut : actif
> Usage : shortlist lisible rapidement des exigences documentaires et techniques encore ouvertes

---

## But

Ce fichier n'est **pas** la baseline complete.

C'est la liste courte de ce qui reste actif, afin que les agents n'aient pas a :

- relire toutes les docs historiques ;
- deviner quelles exigences sont encore ouvertes ;
- consommer des tokens sur des sections deja implementees ou obsoletees.

---

## Statuts autorises

- `open` : exigence reconnue mais non traitee
- `in_progress` : travail commence
- `implemented_unvalidated` : code/documentation modifie mais validation encore partielle
- `validated` : preuve code + validation disponible
- `superseded` : remplace par une exigence plus recente

## Regle d'entretien

Quand une exigence est traitee :

1. mettre a jour son statut ici ;
2. ajouter une entree dans `PRIMARY_SOURCE_DELTA_LEDGER.md` ;
3. si l'exigence change de nature, superseder au lieu de reecrire silencieusement l'historique.
